Forward Pass Math

Exact function used by the AI on every turn

x = 2 planes × 6 rows × 7 columns (mine, opponent)
trunk = ReLU(Conv3×3(x, 64 channels))
repeat 4×: trunk = ReLU(trunk + Conv3×3(ReLU(Conv3×3(trunk))))
policy = SoftmaxLegal(Dense(ReLU(Conv1×1(trunk))))
value = tanh(Dense(ReLU(Dense(ReLU(Conv1×1(trunk))))))
loss = soft solver-policy cross entropy + 0.25 × value MSE
AI move = argmax(legal column) policy[column]

The solver exists only to create labels and to play when selected. The AI player calls the network once and uses no search, win/block rule, or solver fallback.
