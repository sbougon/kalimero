Forward Pass Math

Exact function used by the AI on every turn

input x = [my_piece(0..41), opponent_piece(0..41),
           my_win_now(0..6), opponent_win_now(0..6)]
h1[j] = ReLU(b1[j] + sum(i=0..97) W1[j,i] * x[i])
h2[k] = ReLU(b2[k] + sum(j=0..95) W2[k,j] * h1[j])
logit[c] = b3[c] + sum(k=0..63) W3[c,k] * h2[k]
           + sum(t=0..13) Skip[c,t] * tactical_input[t]
P(column=c) = exp(logit[c]) / sum(legal q) exp(logit[q])
loss = -sum(c=0..6) teacher[c] * log(P(column=c))
AI move = argmax(legal c) P(column=c)

The solver exists only to create labels and to play when selected. The AI player calls the network once and uses no search, win/block rule, or solver fallback.
